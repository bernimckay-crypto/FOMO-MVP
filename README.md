# FOMO MVP

Plataforma digital para descubrir actividades, talleres y comunidades en Santiago.

## 🚀 Ver en vivo

👉 [https://bernimckay-crypto.github.io/FOMO-MVP](https://bernimckay-crypto.github.io/FOMO-MVP)

## 📁 Archivos

- `index.html` — Plataforma completa (MVP funcional)

## ✨ Funcionalidades

- Onboarding personalizado por rol (Usuario / Organizador)
- Catálogo de 9 actividades con filtros
- Sistema de favoritos
- Mapa interactivo
- Asistente IA (Fomito) conectado a Claude API
- Panel organizador con gestión de inscritos y pagos
- Panel admin oculto con analytics y exportación CSV/Excel
- Base de datos conectada a Supabase

## 🔐 Acceso admin

`Ctrl + Shift + A` desde cualquier pantalla, o clic en ⚙ en el footer.

## 🛠 Stack

- HTML + CSS + JavaScript vanilla
- Supabase (base de datos)
- Claude API (chatbot IA)
- GitHub Pages (hosting)
